## 摘要
开放词汇的3D场景理解旨在通过从视觉-语言模型中迁移语义知识，对超出预定义类别的3D场景进行分割。现有方法通过将对齐语言的2D特征提升到3D来推进这一任务，但它们通常依赖与上下文无关的语义表示，未能充分利用物体间关系进行上下文精炼。我们提出RelGraphOV，一种关系感知框架，利用3D场景图增强开放词汇的3D理解。我们的方法从多视角观测构建关系场景图，借助视觉-语言推理推断物体间关系并修剪几何上不可信的连接，无需手动关系标注。为在避免特征干扰的同时聚合关系上下文，我们引入自适应门控双流上下文图注意网络，它分离密集几何特征与语义CLIP嵌入，执行边引导的消息传递，并自适应融合互补语义。一个层次对比目标进一步促进实例级一致性和类别级区分。在ScanNetV2、ScanNet200、ScanNet$++$和Replica上的实验展示了强大的性能和泛化能力。项目页面：https://cxavireh.github.io/relgraphov-projectpage

## Abstract
Open-vocabulary 3D scene understanding aims to segment 3D scenes beyond predefined categories by transferring semantic knowledge from vision-language models. Existing methods have advanced this task by lifting language-aligned 2D features into 3D, yet they often rely on context-independent semantic representations, leaving object relationships underexplored for contextual refinement. We propose RelGraphOV, a relationship-aware framework that uses 3D scene graphs to enhance open-vocabulary 3D understanding. Our method constructs relational scene graphs from multi-view observations by leveraging vision-language reasoning to infer object relationships and prune geometrically implausible connections, without manual relationship annotations. To aggregate relational context while avoiding feature interference, we introduce an Adaptive Gated Dual-Stream Contextual GAT that separates dense geometric features and semantic CLIP embeddings, performs edge-guided message passing, and adaptively fuses complementary semantics. A hierarchical contrastive objective further promotes instance-level consistency and category-level discrimination. Experiments on ScanNetV2, ScanNet200, ScanNet$++$, and Replica demonstrate strong performance and generalization ability. Project Page: https://cxavireh.github.io/relgraphov-projectpage

# 论文总结：RelGraphOV

## 1. 核心问题与研究动机
- **背景**：开放词汇3D场景理解旨在分割超出预定义类别的3D场景，现有方法通过将2D视觉-语言模型（如CLIP）特征提升到3D来迁移语义知识。
- **问题**：这些方法通常依赖**上下文无关的语义表示**，忽略了物体之间的结构关系（例如“窗帘”vs“浴帘”仅靠外观难以区分，但结合与浴缸/马桶的空间关系可自然解决）。
- **目标**：引入3D场景图显式建模物体关系，并利用关系上下文精炼开放词汇语义特征，提升分割精度和泛化能力。

## 2. 方法论：RelGraphOV 框架
### 2.1 核心思想
- 构建“关系感知3D场景图”，通过多视图视觉-语言推理自动推断物体间关系（无需手动标注）。
- 提出**自适应门控双流上下文图注意网络（Adaptive Gated Dual-Stream Contextual GAT）**，分离密集几何特征（LSeg）和语义CLIP特征，执行边引导消息传递，并通过可学习门控自适应融合互补语义。
- 设计**层次对比学习目标**，同时促进实例级一致性和类别级判别，防止语义漂移。

### 2.2 关键技术细节
- **场景图构建**：
  - 输入RGB-D序列或点云，通过MaskClustering等获得类无关的3D物体提案作为节点。
  - 基于空间近邻初始化候选边，利用VLM（qwen-vl-max）对每个候选边进行链式推理，判断关系类型，滤除几何上不可信连接（返回"none"则修剪）。
  - 对有效边提取主谓宾三元组，经过CLIP文本编码器形成结构化的边特征。
- **节点特征初始化**：
  - 主流：几何特征（中心+包围盒经傅里叶编码） + 密集LSeg特征。
  - 辅助流：几何特征 + 实例级CLIP特征。
- **自适应门控双流GAT**：
  - 全局上下文聚合：引入虚拟超节点连接所有真实节点，通过标准GAT获得全局特征，再通过交叉注意力注入主流。
  - 边引导消息传递：在注意力机制中将边特征拼接到Key中，使注意力权重受关系语义调控。
  - 可学习门控融合：Gate = σ(MLP([主流状态 ‖ 辅助流状态]))，主流更新时门控注入辅助流特征；辅助流保持不变，避免干扰。
- **损失函数**：
  - 层次对比损失 L_hie：自一致性、同类一致性、异类排斥。
  - 双重对齐损失 L_reg（主流与VLM标注对齐同时保持LSeg先验）、L_aux（辅助流与VLM标注对齐同时保持CLIP先验），防止灾难性遗忘。

## 6. 主要结论与发现
- RelGraphOV在**ScanNetV2**上mIoU 58.4%，超越OV3D（57.3%）等；在**ScanNet200**上mIoU 14.5%，领先次优方法（Mosaic3D 12.4%）约2个点。

- **零样本泛化**：在ScanNet$++$上mIoU 20.9%（OpenScene-2D仅13.3%）；在Replica上mIoU 22.7%（OpenNeRF未零样本，需微调；零样本基线最高12.9%）。

## 7. 优点
- **无需人工关系标注**：利用VLM链式推理自动构建关系场景图，成本低、可扩展。
- **双流解耦设计**：有效缓解几何噪声与语义特征之间的干扰，保留各自优势。
- **层次对比+双重对齐损失**：在吸收VLM监督的同时保持原始特征先验，避免语义漂移。
- **强泛化性**：跨数据集零样本效果突出，尤其对长尾和跨域场景鲁棒。
